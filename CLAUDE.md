# 프로젝트 설정

## Python 가상환경

이 프로젝트는 `venv` 폴더의 Python 가상환경을 사용합니다.

**모든 Python 관련 명령어는 반드시 venv를 사용해야 합니다:**

- Python 실행: `.\venv\Scripts\python` (PowerShell) 또는 `venv/Scripts/python` (bash)
- pip 실행: `.\venv\Scripts\pip` (PowerShell) 또는 `venv/Scripts/pip` (bash)
- 스크립트 실행: `.\venv\Scripts\python script.py`
- 패키지 설치: `.\venv\Scripts\pip install <package>`

`python`, `pip` 명령어 단독 사용 금지 — 항상 venv 경로를 명시할 것.
